# 🎯 Trivia Maker Website

A simple and customizable web app for creating, managing, and playing trivia quizzes. Whether you're hosting a game night, building a classroom activity, or just having fun, this tool makes it easy to generate engaging trivia content.

---

## 🚀 Features

* 🧠 Create custom trivia questions with multiple-choice answers
* 📂 Organize quizzes by categories or topics
* ▶️ Play quizzes in an interactive format
* 💾 Save and edit existing quizzes
* 📊 Track scores and performance
* 🌐 Responsive design for desktop and mobile

---

## 🛠️ Tech Stack

* Frontend
* Backend

> You will need Visual Studio Code (VS Code) to run and edit the project.

---

## 🎮 Usage

1. Create a new quiz by clicking **"New Quiz"**
2. Add questions, answer choices, and mark the correct answer
3. Save your quiz
4. Start playing and test your knowledge!
